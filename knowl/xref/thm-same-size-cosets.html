Theorem 4.3.8. Coset "Size".

A left (or right) coset in a finite group contains the same number of elements as the subgroup that generated the coset.
Proof.
Suppose \(H \lt G\) (i.e. \(H\) is a subgroup of \(G\)). Further, let \(H=\left\{h_1, h_2, h_3, \ldots, h_m\right\}\text{.}\) Now we can form a left coset, \(aH\text{,}\) such that \(aH=\left\{ah_1, ah_2, ah_3, \ldots, ah_m\right\}\text{.}\) We can claim that all elements of this coset are unique since if not, we must have that \(ah_i=ah_j\text{,}\) for some \(i\) and \(j\) with \(i \neq j\text{.}\) But this would mean that \(a^{-1}ah_i=a^{-1}ah_j \Rightarrow\) \(eh_i=eh_j\text{,}\) and thus \(h_i=h_j\) and so \(ah_i \neq ah_j\text{.}\)
