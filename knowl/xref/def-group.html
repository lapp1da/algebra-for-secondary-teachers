Definition 4.3.1.

A group is a set, \(G\) along with a binary operation, \(*\text{,}\) such that if \(a,b,c \in G\) the following properties hold.
Closure : If \(a,b \in G\text{,}\) then \(a*b \in G\text{.}\)
Associativity : If \(a,b,c \in G\text{,}\) then \(a*\left(b*c\right)=\left(a*b\right)*c\text{.}\)
Identity : There exists an element, \(e \in G\text{,}\) (called the identity) such that \(e*a=a*e=a\) for all \(a \in G\text{.}\)
Inverses : For all \(a \in G\text{,}\) there exists an inverse element, denoted \(a^{-1}\text{,}\) such that \(a*a^{-1}=a^{-1}*a=e\text{.}\)
